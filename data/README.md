# Data

Datasets used in *Osang'ir et al. (2026)*.

| Dataset | Where it lives | Notes |
|---------|----------------|-------|
| **CLL** (methylation + drugs) | [`CLL/`](CLL/) | Loaded from the **MOFAdata** R package — see `CLL/README.md` |

## Large precomputed objects (NOT in this repo)

The following intermediate/result objects were used during the analysis but
**exceed GitHub's 100 MB per-file limit** and are therefore not committed. They
can be regenerated by the scripts, or shared via Git LFS / an archive (e.g.
Zenodo) on request.

| File (in the working tree) | Size | What it is |
|----------------------------|------|------------|
| `publishable code/spl_spz_alpha_monitor_cll.rds` | 5.5 GB | FABIA sparsity-grid monitoring (Fig 6) |
| `publishable code/final_benchmark_fabia_cutoff.rds` | 129 MB | FABIA cutoff benchmark (Fig 7) |
| `publishable code/data_4_final_benchmark_fabia_cutoff.rds` | 109 MB | Input to the above |
| `*.RData` session images (paper_II, rad_data, test_sim, …) | 0.2–11 GB | R workspace snapshots |
| `test_sim/.../sim_omicResults_*_sigma_*.rds` | ~8 MB each | Per-noise-level simulation outputs (Fig 11 + Suppl) |

> To version large files on GitHub, enable [Git LFS](https://git-lfs.com): e.g.
> `git lfs track "*.rds"`. For permanent archival with a DOI, deposit them on
> [Zenodo](https://zenodo.org) and link here.
